# 🛒 Lukas - Smart Shopping Copilot

**PWA móvil optimizada para el mercado colombiano**

Lukas es tu asistente inteligente de compras que te ayuda a ahorrar comparando precios en tiempo real entre supermercados como D1, Ara, Éxito, Carulla, Jumbo y Olímpica, mientras gestionas presupuestos compartidos con tu familia.

---

## ✨ Características Principales

### 🧠 El Comparador Mágico (The Brain)
- Comparación de precios en tiempo real entre múltiples tiendas
- **IA Optimizer**: Reorganiza tu lista automáticamente para máximo ahorro
- Visualización clara del mejor precio por producto
- Interface Bento Grid con Glassmorphism

### ❤️ Presupuestos Compartidos (The Heart)
- Anillos de energía dinámicos estilo Apple Watch
- Visualización de miembros activos del grupo
- Semaforización ambiental según salud financiera
- Registro de gastos colaborativo

### ✨ Ocasiones Mágicas (The Magic)
- Kits predefinidos para eventos especiales:
  - 🥩 Asado Familiar
  - 🍷 Cena Romántica
  - 🧺 Picnic Familiar
  - 🥞 Brunch Dominguero
  - 🎂 Fiesta de Cumpleaños
  - 🍖 Parrillada Premium
- Listas de compras optimizadas por ocasión

### 🎨 Lukas - El Orbe Dinámico
Mascota visual que cambia según el contexto:
- 🔵 **Neutral (Azul)**: Navegación tranquila
- 🔷 **Scanner (Turquesa)**: Comparando precios
- 🟡 **Success (Dorado)**: ¡Ahorro encontrado!
- 🟠 **Alerta (Coral)**: Presupuesto cerca del límite

---

## 🚀 Tecnologías

- **React 18** con TypeScript
- **React Router v7** (Data Mode)
- **Tailwind CSS v4**
- **Motion** (Framer Motion) para animaciones fluidas
- **Lucide React** para iconografía
- **Context API** para gestión de estado global

---

## 📱 Optimización Móvil

- Diseño responsive optimizado para 390px (móvil)
- Bottom Navigation para fácil navegación con una mano
- Touch-friendly (todos los botones mínimo 44x44px)
- Animaciones a 60fps con GPU acceleration
- PWA ready con manifest.json

---

## 🎯 Navegación

La aplicación cuenta con 4 pantallas principales:

1. **Home** (`/`) - Pantalla de bienvenida con navegación rápida
2. **The Brain** (`/brain`) - Comparador de precios
3. **The Heart** (`/heart`) - Gestión de presupuestos compartidos
4. **The Magic** (`/magic`) - Feed de ocasiones especiales

## 🎨 Sistema de Diseño

### Paleta de Colores
- **Backgrounds**: `#0a0f16` → `#101922`
- **Primary Blue**: `#258CF4`
- **Cyan Bright**: `#00F5FF`
- **Success Green**: `#2ECC71`
- **Warning Orange**: `#FB923C`
- **Alert Red**: `#FF5757`

### Tipografía
- **Font**: Outfit (Google Fonts)
- **Pesos**: 300, 400, 500, 600, 700, 800

### Efectos Visuales
- Glassmorphism con `backdrop-blur`
- Gradientes radiales para backgrounds
- Sombras profundas con colores temáticos
- Animaciones suaves con easing `[0.4, 0, 0.2, 1]`
